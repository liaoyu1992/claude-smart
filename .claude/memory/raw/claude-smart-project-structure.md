The project is located at `C:/Users/liaoyu/work/cluade-smart/`. The memory system utilizes a `.claude` directory containing `bin` (for scripts like `auto-evolve.py`, `auto-analyze-instincts.py`, `extract_memory.py`), `hooks` (for shell scripts like `observe.sh`, `stop.sh`, `session-inject.sh`), `memory` (for Python modules like `inject_memory_context.py`, `embed.py`, `vector_store.py`), and `data/observations` (storing `observations.jsonl`).
